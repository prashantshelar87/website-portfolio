# Prashant Shelar Portfolio

This is a Vite + React portfolio app.

## Run locally

Prerequisites: Node.js

1. Install dependencies:

   npm install

2. Start dev server:

   npm run dev

3. Build for production:

   npm run build

## Deploy to GitHub Pages

This project is configured to be deployed as a GitHub project page. The Vite `base` is set to `/prashant-shelar-portfolio/` in `vite.config.ts`.

1. Install dev dependencies (if not already):

   npm install

2. Deploy:

   npm run deploy

Note: The `deploy` script uses `gh-pages` and publishes the `dist` folder to the `gh-pages` branch. Make sure your repository name matches `prashant-shelar-portfolio` or update `base` in `vite.config.ts` accordingly.
